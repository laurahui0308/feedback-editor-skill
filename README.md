# Feedback Editor Skill

Single-file visual annotation editor for any web project. Click-to-pin, type feedback, save.

## Usage

```
/plugin marketplace add <your-username>/feedback-editor
```

Then mention `feedback-editor` in any project to install it.

## How it works

Adds a pencil icon to the page. Click it to enter edit mode:
- Click any element to place a numbered annotation pin
- Type feedback in the popup, press Enter to confirm
- Press Delete to remove a pin
- Click Save to export all annotations

## Files

- `feedback-editor.js` — zero-dependency vanilla JS, just add a `<script>` tag
- `SKILL.md` — AI-readable install instructions

## Browser Support

All modern browsers. No build step. No npm install.
